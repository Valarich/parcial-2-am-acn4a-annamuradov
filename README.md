# Mis hábitos

## Descripción de la pantalla

La pantalla desarrollada pertenece a una aplicación para organizar hábitos diarios.

El objetivo de esta pantalla es permitir que el usuario visualice una presentación simple de la app y pueda agregar hábitos a una lista mediante un botón.

## Elementos implementados

- Título principal: “Mis hábitos”.
- Imagen representativa relacionada con listas y organización de hábitos.
- Descripción breve de la funcionalidad.
- Botón “Agregar hábito”.
- Lista dinámica donde se agregan hábitos mediante Java.

## Funcionalidad

Al presionar el botón “Agregar hábito”, la aplicación agrega dinámicamente un nuevo TextView a la pantalla con el texto del hábito.

Ejemplo:

“Hábito 1: tomar agua”

## Recursos utilizados

La pantalla utiliza:

- ConstraintLayout.
- LinearLayout vertical.
- Button.
- TextView.
- ImageView.
- Variables de String en `strings.xml`.
- Variables de dimensiones en `dimens.xml`.
- Variables de colores en `colors.xml`.
